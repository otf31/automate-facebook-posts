# Automate facebook posts

Monorepo to store **automate facebook posts** related applications.

## Aplications
1. tui: Textual user interface to publish posts. Build with Python 3 and Textual.
2. cli: Command line interface to publish posts. Build with Python 3 and Typer.
3. backend: API related to user subscriptions. Build with Python 3 and FastAPI.
